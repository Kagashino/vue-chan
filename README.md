# Vue Chan
An extremely simple [petite-vue](https://github.com/vuejs/petite-vue) template.  
"Chan" means "ちゃん"

## Usage

### fetch
```shell
git clone https://github.com/Kagashino/vue-chan.git
```

### install
```
yarn install
```

### run
```shell
yarn dev
```

### build
```shell
yarn build
```
